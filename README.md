# LAYR Submission Guidelines

- Submission is done by creating a pull request in this repository containing all relevant files

- Your submission should be placed in a separate directory within this repository, named as `institution-team` (e.g., `rub-1`)

## Minimum requirements for submission include:
- A `README.md` file describing the directory / file structure of your submission
- A `src` directory containing all of your RTL sources
- A `docs` directory containing any supporting documentation

## Supplementary material
Besides these required aspects, you can include any data which might help reviewers understand
or analyze your submission. Make sure to highlight the structure of your supplementary material in the README file.
